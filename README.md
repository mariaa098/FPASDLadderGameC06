Ladder Game merupakan permainan papan berbasis giliran di mana pemain harus mencapai tile 64 berdasarkan hasil lemparan dadu. Versi ini dikembangkan dengan pendekatan algoritma dan struktur data, sehingga sistem dapat menentukan jalur paling optimal menuju kemenangan. Permainan ini memodelkan papan sebagai graph, di mana setiap tile adalah node dan setiap kemungkinan lemparan dadu merupakan edge. Fitur ladder dan bonus poin memengaruhi jalur yang diambil pemain.
